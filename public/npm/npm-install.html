npmをインストールする

2021-08-22

概要

公式サイト：ダウンロード | Node.js

Node.jsをインストールするとnpmも一緒に入る。

上記公式サイトからインストーラをダウンロードしてインストールするか、Homebrewなどのパッケージマネージャを使う。

ここではHomebrewを使って→インストールしたnodebrewで→Node.jsをインストールする。

備考：既にNode.jsが入っているがnodebrewで管理したくなった場合
【Mac版】node.jsのアンインストールと再インストール手順メモ - Qiita

Homebrewをインストールする

公式サイト：macOS（またはLinux）用パッケージマネージャー — Homebrew

以下をターミナルにコピペしてEnter。

% /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

nodebrewをインストールする

参考：Node.jsの管理をHomebrewからnodebrewに変える - Qiita

Homebrewを使ってnodebrewをインストールする

% brew install nodebrew

nodebrewの初期設定を行う

% /usr/local/opt/nodebrew/bin/nodebrew setup_dirs

デフォルトのシェルを確認する

参考：【zsh, bash】macでPATHを通す方法 - not found と出てしまったら - Qiita

 % echo $SHELL
/bin/zsh

デフォルトのシェルがzshだった場合、~/.zshrcファイルを開くか、まだない場合は作成し、以下を追記する。

.zshrc

export PATH=$HOME/.nodebrew/current/bin:$PATH

変更を反映する

% source ~/.zshrc

ヘルプを表示してnodebrewが使えるか確認する

% nodebrew help

Node.jsをインストールする

nodebrewを使ってNode.jsの最新版をインストールする。

% nodebrew install-binary latest

nodebrewでインストールしたNode.jsのバージョンを確認する。

% nodebrew ls
v16.7.0
 
current: none

使うバージョンを設定する。最新版を使う場合

% nodebrew use latest

インストールしたNode.jsのバージョン一覧と現在使っているバージョンを表示する

% nodebrew ls
v16.7.0
current: v16.7.0

nodeコマンドが使えるかバージョンを表示して確認

% node -v
v16.7.0

npm自体のバージョンの確認

% npm -v
7.20.3
