gitを使う

2020-03-29

参考：Getting Started - First-Time Git Setup
デプロイ！

gitをインストールする

gitをダウンロードする

公式サイト：https://git-scm.com

そのOSで初めてgitを使うとき

自分のユーザ名とメールアドレスを設定する

$ git config --global user.name "Your Name"
$ git config --global user.email youremail@address.com

設定した情報と設定ファイルのある場所を確認する

$ git config --list --show-origin

gitを有効にする

プロジェクトディレクトリに移動する

$ cd
$ cd MyProject

gitを有効にする

$ git init

変更を記録したくないファイルを無視させる

参考：.gitignore の書き方

プロジェクトディレクトリ直下に.gitignoreという名前のテキストファイルを作る。

  • MyProject
    • .gitignore

.gitignore

.DS_Store
.sass-cache/
vendor/

ファイルの状態を確認する

$ git status

未追跡(変更を記録することにしていない)ファイルが赤色で表示される。

新しいファイルの追跡を開始する

$ git add -A

オプション-A allの略。すべてのファイル

$ git status

追跡しているファイルが緑色で表示される。

はじめてのコミット

$ git commit -m "My first commit"
$ git commit -m "どこをどう変更したのか"
