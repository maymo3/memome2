Sassを使う

2020-03-28

参考：sass / ruby-sass

Rackに対応しているWebフレームワーク(sinatra含む)では以下の方法でgem 'sass'を使うことができる。

gemをインストールする

まずはgemをインストールする。

$ gem install sass

bundlerを使用している場合は、

Gemfile

gem 'sass'

をGemfileに追記して

$ bundle install

config.ruに設定を書く

config.ru

require 'sass/plugin/rack'
use Sass::Plugin::Rack

sassを置くディレクトリを作る

  • My Project
    • public
      • stylesheets
        • sass
          • style.sass

public/stylesheets/sassに置いたすべてのsassファイルが、リクエストの度にpublic/stylesheets以下のcssファイルに変換される。
例えばpublic/stylesheets/sassにstyle.sassを作った場合、public/stylesheetsにstyle.cssが生成される。
