bundlerを使う

2020-03-25

bundlerをインストールする

$ sudo gem install bundler --no-doc

プロジェクトディレクトリでbundlerを使うことを表明する

$ bundle init

プロジェクトディレクトリにGemfileが生成される

  • MyProject
    • app.rb
    • Gemfile

Gemfileを編集する

の部分をGemfileに追記する

Gemfile

# frozen_string_literal: true
 
source "https://rubygems.org"
 
git_source(:github) {|repo_name| "https://github.com/" }
\# gem "rails"
gem 'sinatra'
gem 'sinatra-contrib'
gem 'slim'
gem 'sass'

gemをインストールする

gemのインストール場所を設定する

$ bundle config set path 'vendor/bundle'

gemをインストールする

$ bundle install

指定したvendor/bundleディレクトリが作られ、その配下にgemがインストールされる。
インストールしたgem一覧を記録するGemfile.lockが生成される。

  • MyProject
    • app.rb
    • Gemfile
    • Gemfile.lock
    • vendor

bundlerを使ってサーバを起動する

※ipアドレスは一例

$ bundle exec ruby app.rb -o 192.168.33.10
